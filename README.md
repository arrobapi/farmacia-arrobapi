# Farmacia arrobaPi

Este repositorio contiene una página web estática para **Farmacia arrobaPi**.

## Aviso importante

Este es un **proyecto ficticio** creado únicamente con fines educativos. La web, el contenido y la farmacia representada forman parte de un ejercicio para practicar con un alumno la creación de páginas web modernas, sencillas y funcionales.

No se trata de una web oficial de una farmacia real ni debe utilizarse como fuente de información sanitaria profesional.

## Objetivo del proyecto

El objetivo principal es practicar conceptos básicos de desarrollo web, como:

- Estructura HTML semántica.
- Estilos CSS modernos y responsive design.
- Creación de una landing page funcional.
- Organización de secciones habituales de una web corporativa.
- Uso de un mapa interactivo con **Leaflet** y **OpenStreetMap**.
- Preparación de una página estática para publicarla con **GitHub Pages**.

## Contenido de la web

La página incluye:

- Cabecera con logotipo ficticio.
- Presentación de la farmacia.
- Horario de apertura.
- Sección de servicios habituales.
- Zona de consejos farmacéuticos.
- Ubicación con mapa interactivo.
- Diseño adaptable para móvil, tablet y escritorio.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Leaflet
- OpenStreetMap

## Publicación con GitHub Pages

Para publicar este proyecto en GitHub Pages:

1. Sube los archivos del proyecto a un repositorio de GitHub.
2. Accede a **Settings > Pages**.
3. En la sección **Build and deployment**, selecciona la rama principal, normalmente `main`.
4. Selecciona la carpeta raíz del repositorio.
5. Guarda los cambios.
6. GitHub generará una URL pública para visualizar la web.

## Uso educativo

Este proyecto puede utilizarse como base para explicar o practicar:

- Cómo crear una página web desde cero.
- Cómo estructurar el contenido de una web sencilla.
- Cómo aplicar estilos visuales coherentes.
- Cómo hacer una página responsive.
- Cómo insertar un mapa gratuito sin usar servicios de pago.
- Cómo publicar una web estática en internet mediante GitHub Pages.
